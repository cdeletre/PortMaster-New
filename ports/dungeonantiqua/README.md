## Language
By default the game will be in Japanese, you can change this in the config menu.

## Controls

| Button | Action |
|--|--| 
|DPAD|Move, cursor selection|
|A|OK|
|B|Cancel|
|X|Open Menu|
|Y|Help / Reset (hold 2 seconds)|
